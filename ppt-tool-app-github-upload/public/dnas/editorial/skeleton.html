FRW-A · EDITORIAL · PG. 92
Q1 — 利基策略

高品牌 + 单市场

深耕一个市场,以品牌为王。代表 Lotus / 蔚来。

Q2 — 拐点机会 ★

高品牌 + 全球化

真正的"中国 Tesla" — 全球品牌溢价 + 规模化。

Q3 — 内卷困境

低品牌 + 单市场

本土价格战 / 利润压缩 / 难以出海。

Q4 — 量价博弈

低品牌 + 全球化

规模出海但低价 / 品牌资产积累慢 / 长期路径不清。

— FRAMEWORK · 四层结构

出海能力的四层结构。

01
政策层 Policy

国家战略 / 出口补贴 / 出口管制 — 决定出海"能不能做"的前提。

02
供应链层 Supply Chain

动力电池 / 电驱 / 电控三大件全球领先 — 中国 NEV 出海最深的护城河。

03
品牌层 Brand

海外品牌资产 / 本地团队建设 — 从 OEM 到独立运营的关键过渡。

04
用户层 Consumer

海外用户认知转变 / 复购率提升 — 出海是否"真正立住"的最终检验。
